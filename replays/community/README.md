# Community Bundles

This directory holds replay bundles contributed by the community.

Bundles here have been anonymized and reviewed for sensitive data.

## How to contribute

1. Capture the failure with `frl capture`
2. Anonymize: `frl anonymize --bundle my-bundle.json -o my-bundle-clean.json`
3. Validate: `frl validate --bundle my-bundle-clean.json`
4. Open a PR or use the **Community Bundle Submission** issue template

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for the full guide.
